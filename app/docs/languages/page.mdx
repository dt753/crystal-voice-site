# Языки распознавания

Crystal Voice использует Whisper — многоязычную модель распознавания речи.
Поддерживаются десятки языков, включая русский, английский, украинский и другие.

## Как выбрать язык

1. Открой главное окно Crystal Voice
2. Перейди в **Настройки → Язык**
3. Выбери нужный язык из списка

## Поддерживаемые языки

| Язык | Код |
|---|---|
| Русский | `ru` |
| Английский | `en` |
| Украинский | `uk` |
| Немецкий | `de` |
| Французский | `fr` |
| Испанский | `es` |
| Польский | `pl` |
| Турецкий | `tr` |

И многие другие — Whisper поддерживает более 50 языков.

## Автоопределение языка

Если выбрать **Auto**, Whisper сам определит язык по первым секундам речи.
Это удобно если ты переключаешься между языками.

> **Совет:** для лучшего качества распознавания выбирай язык вручную —
> это быстрее и точнее чем автоопределение.
