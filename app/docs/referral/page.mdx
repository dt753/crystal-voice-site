# Реферальная программа

## Как это работает

Каждый пользователь Crystal Voice получает уникальный реферальный код.

1. Зайди в [Аккаунт](/account) и скопируй свою реферальную ссылку
2. Поделись ею с другом
3. Друг регистрируется по ссылке или вводит код в своём аккаунте
4. **Друг получает +30 дней тарифа Старт**, ты — **💎 +200 кристаллов**

## Ограничения

Максимум **5 рефералов** на один аккаунт. Чем выше тариф, тем больше кристаллов за каждого друга:

- Старт: 💎 +200 кристаллов за реферала
- Pro: 💎 +300 кристаллов за реферала

## Что такое кристаллы?

Кристаллы 💎 — внутренняя валюта Crystal Voice. 100 кристаллов ≈ 30 минут транскрипции.
Новые пользователи получают **💎 100 кристаллов** при регистрации.

## Где ввести чужой код

В разделе [Аккаунт → Реферальный код](/account) введи код и нажми **Применить**.

> Код можно применить только один раз для одного аккаунта.
