# Реферальная программа

## Как это работает

Каждый пользователь Crystal Voice получает уникальный реферальный код.

1. Зайди в [Аккаунт](/account) и скопируй свой код
2. Поделись им с другом
3. Друг вводит твой код в своём аккаунте
4. **Оба получают +30 дней Premium** автоматически

## Без ограничений

Количество рефералов не ограничено — чем больше друзей, тем больше бесплатного Premium.

## Где ввести чужой код

В разделе [Аккаунт → Реферальный код](/account) введи код и нажми **Применить**.

> Код можно применить только один раз для одного аккаунта.
