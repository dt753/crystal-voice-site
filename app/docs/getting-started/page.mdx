# Быстрый старт

Crystal Voice — это приложение для голосового ввода текста на Windows и Mac.
Нажал горячую клавишу — говори — текст появился в активном окне.

## Установка

1. Перейди на страницу [Скачать](/download)
2. Выбери версию под свою платформу (`.exe` для Windows, `.dmg` для Mac)
3. Установи приложение

## Первый запуск

После установки Crystal Voice появится в системном трее.

1. Кликни на иконку в трее — откроется главное окно
2. Войди в аккаунт (или зарегистрируйся)
3. Разреши доступ к микрофону при первом запросе

## Используй голосовой ввод

**Хоткей по умолчанию:** двойной тап `Caps Lock`

```
Caps Lock × 2 → запись начата
говори...
Caps Lock × 2 → запись остановлена → текст вставлен
```

Текст вставляется в то место, где находится курсор — в браузере, мессенджере, редакторе.

## Смена языка

В разделе **Настройки → Язык** выбери нужный язык распознавания.
По умолчанию стоит **Русский**.
