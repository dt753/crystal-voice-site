# Установка

## Windows

1. Перейди на страницу [Скачать](/download) и скачай файл `Crystal-Voice-Setup.exe`
2. Запусти установщик — Windows SmartScreen может показать предупреждение, нажми **Всё равно запустить**
3. Дождись завершения установки — приложение запустится автоматически
4. Иконка Crystal Voice появится в системном трее (правый нижний угол)

**Системные требования:** Windows 10 / 11 (x64)

## macOS

1. Перейди на страницу [Скачать](/download) и скачай файл `Crystal-Voice.dmg`
2. Открой `.dmg` и перетащи приложение в папку **Applications**
3. При первом запуске macOS попросит подтвердить запуск — зайди в **Настройки → Конфиденциальность и безопасность** и нажми **Открыть**
4. Разреши доступ к микрофону при первом запросе

**Системные требования:** macOS 12 и выше (Intel + Apple Silicon)

## После установки

1. Кликни на иконку в трее — откроется главное окно
2. Войди в аккаунт или зарегистрируйся на странице [Аккаунт](/account)
3. Дважды нажми `Caps Lock` — и можно говорить

## Обновление

Crystal Voice обновляется автоматически при наличии новой версии. Все релизы также доступны на [GitHub Releases](https://github.com/dt753/Voice-Typer/releases).

## Удаление

**Windows:** Панель управления → Программы → Crystal Voice → Удалить

**macOS:** Перетащи приложение из папки Applications в корзину
